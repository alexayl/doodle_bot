Eraser path density

CV detects dark regions (smudges) on the board and generates targeted erase paths. More efficient than full board erase.

How It Works
  • Captures calibrated board image
  • Detects dark regions using threshold
  • Filters by minimum area to ignore specs
  • Generates horizontal raster paths for each smudge
  • Optimizes order by distance (nearest first)
  • Executes with CV-based path correction
Tips
  • Lower darkness threshold for light smudges
  • Increase min area to ignore dust specs
  • Decrease line spacing for thorough erasing
  • Ensure board calibration is locked first
